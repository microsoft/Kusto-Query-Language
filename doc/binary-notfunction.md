# binary_not()

Returns a bitwise negation of the input value.

```kusto
binary_not(x)
```

**Syntax**

`binary_not(`*num1*`)`

**Arguments**

* *num1*: numeric 

**Returns**

Returns logical NOT operation on a number: num1.
