# atan()

Returns the angle whose tangent is the specified number (the inverse operation of [`tan()`](tanfunction.md)) .

## Syntax

`atan(`*x*`)`

## Arguments

* *x*: A real number.

## Returns

* The value of the arc tangent of `x`
